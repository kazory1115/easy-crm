## 📋 系統功能

### 1. **報價管理系統** (首頁)
**路徑**: `/` (http://localhost:5175/)

**功能**:
- ✅ 建立新報價單
- ✅ 填寫客戶資訊（客戶名稱、聯絡電話、報價單號、日期）
- ✅ 從項目範本選擇項目（一般項目 / 主機資料）
- ✅ 自訂填寫項目
- ✅ 自動計算總金額
- ✅ **儲存報價單**（新功能！）
- ✅ 列印報價單
- ✅ 匯出 Word 文件

**使用流程**:
1. 填寫客戶基本資訊
2. 點擊「新增項目」選擇項目類型
3. 從下拉選單選擇範本項目，或直接填寫
4. 調整數量、單價等資訊
5. 點擊「儲存報價單」儲存
6. 可選擇列印或匯出 Word

---

### 2. **項目範本管理** (Add)
**路徑**: `/add` (http://localhost:5175/add)

**功能**:
- ✅ 管理一般項目範本
  - 新增、編輯、刪除一般項目
  - 設定品名規格、數量、單位、單價

- ✅ 管理主機資料範本
  - 新增、編輯、刪除主機資料
  - 設定完整的主機規格（主吊馬力、吊速、橫行馬力、橫速、縱行馬力、縱速）

**使用流程**:
1. 切換分頁（一般項目 / 主機資料）
2. 填寫項目資訊
3. 點擊「新增」按鈕
4. 可隨時編輯或刪除已建立的範本

---

### 3. **歷史報價單** (History)
**路徑**: `/history` (http://localhost:5175/history)

**功能**:
- ✅ 查看所有已儲存的報價單
- ✅ 顯示報價單摘要（客戶名稱、日期、總金額、項目摘要）
- ✅ 查看報價單完整詳情
- ✅ 刪除不需要的報價單
- ✅ 顯示建立/更新時間

**使用流程**:
1. 進入歷史報價單頁面
2. 瀏覽所有已儲存的報價單
3. 點擊「查看」按鈕檢視詳細內容
4. 點擊「刪除」按鈕移除報價單

## 💾 資料儲存方式

### LocalStorage (目前使用)
- 所有資料儲存在瀏覽器的 LocalStorage 中
- 資料不會遺失（除非清除瀏覽器資料）
- **適合**: 單機使用、快速開發、測試

### JSON 檔案結構
系統已建立完整的 JSON 資料結構，方便未來擴充：

```
src/data/
├── items.json      # 一般項目範本
├── hosts.json      # 主機資料範本
└── quotes.json     # 報價單資料
```

## 🔧 技術架構

### 資料管理工具
**檔案**: `src/utils/dataManager.js`

**提供的功能**:
- `getItems()` - 取得一般項目
- `addItem(item)` - 新增一般項目
- `updateItem(id, updates)` - 更新一般項目
- `deleteItem(id)` - 刪除一般項目
- `getHosts()` - 取得主機資料
- `addHost(host)` - 新增主機資料
- `updateHost(id, updates)` - 更新主機資料
- `deleteHost(id)` - 刪除主機資料
- `getQuotes()` - 取得報價單
- `saveQuote(quote)` - 儲存報價單
- `updateQuote(id, updates)` - 更新報價單
- `deleteQuote(id)` - 刪除報價單
- `exportAllData()` - 匯出所有資料（備份）
- `importAllData(data)` - 匯入資料（還原）
- `resetAllData()` - 重置所有資料

### 自動初始化
系統啟動時會自動初始化 LocalStorage，載入 JSON 預設資料。
